# Mis Algoritmos Favoritos

Aquí tienes un ejemplo de un 3x3 normal:
[R U R' U'] = 3x3

Ahora, un movimiento para Piraminx:
[U L' R L] = pyraminx

Y aquí un ejemplo de Clock:
[Uu/Dd] = clock

También puedes usar 4x4:
[Rw U Rw'] = 4x4

¡Y listo! El reproductor detectará automáticamente el tipo de cubo y el algoritmo.
